Importação de alunos.

Para realizar a importação, o arquivo XLSX deve seguir o modelo abaixo:

  • Nome: A primeira coluna deve conter o nome completo do aluno.
  • Telefone: A segunda coluna deve conter o número de telefone do aluno.
  • E-mail: A terceira coluna deve conter o endereço de e-mail do aluno.

Certifique-se de que o arquivo não contenha colunas adicionais e que os dados estejam corretamente organizados conforme o modelo.
